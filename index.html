RiftReframe

An automatic highlight capture and intelligent clipping tool for League of Legends, built on the Overwolf platform.

Core Features

Smart Highlight Capture

Captures kill, death, and assist events through Overwolf Game Event Provider (GEP), combined with visual analysis to determine battle status in real-time. Automatically backtracks to find the true start of combat and intelligently identifies when the fight ends, capturing complete highlight moments rather than fixed-length clips.

Vertical Smart Crop (9:16)

Lightweight local vision algorithm enabling one-click post-match cropping of highlights into 9:16 vertical videos, optimized for short-video platforms.
